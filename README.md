# Hangman Game

## Description
This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship. The player guesses a randomly selected word one letter at a time. The game gives six attempts to guess the correct word.

## Features
- Random word selection
- Letter-by-letter guessing
- Six chances to guess the word
- Displays guessed letters and remaining attempts
- Win and lose messages

## Technologies Used
- Python 3

## Concepts Used
- Random module
- While loop
- If-else statements
- Lists
- Strings
- User input/output

## How to Run
1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:
